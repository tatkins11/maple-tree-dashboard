# Write-Ups UI Smoke Checklist

- Open the dashboard and navigate to `Write-Ups`.
- Confirm the selected week shows two Maple Tree game cards in schedule order.
- In `Pregame`, choose a projection season, leave the full active roster selected, and generate the write-up.
- Confirm the recommended lineup table renders and the Markdown output appears in the text area.
- Confirm the Markdown download button saves a `.md` file with the selected season and week in the filename.
- Switch to a bye week or one-game week and confirm generation is disabled with a clear warning.
- In `Postgame`, upload one screenshot for each game and confirm both images render inline.
- Enter manual scores for any game without stored results, then generate the recap.
- Confirm the recap includes `Weekly Result Summary`, `Game 1 Recap`, `Game 2 Recap`, `Who Played Well`, `Week MVP`, `Where to Improve Next Week`, and `Milestone/Record Context`.
- Confirm the postgame Markdown download button saves a `.md` file successfully.
